# 운영 체크리스트 (코드 외 · 배포 후)

ROI #6 — 엔진 입력이 비면 측정·신뢰 UI가 무의미하다. **사람이 prod에서 실행.**

## 배포 직후 (필수)

- [ ] 관리자 → **데이터 갱신** 1회 클릭  
  → 5년 가격 + DART 재무·기업개황 자동 백필 (첫 클릭 수 분, CLI 불필요)
- [ ] `/api/accuracy` 또는 관리자>엔진 **실측 성과** — `coverage.dates` 증가 확인
- [ ] 시그널 탭 신뢰 스트립에 **누적중** 배지·성숙 N/20 표시 확인

## 비밀·알림

- [ ] DART 키가 채팅/로그에 평문 노출된 적 있으면 **재발급**(무료) 후 prod `.env` 교체
- [ ] 텔레그램 봇 `token` / `chat_id` prod `.env` 확인(알림 전송 테스트 1건)
- [ ] `ANTHROPIC_API_KEY` · `ADMIN_EMAILS` · `APP_ENV=prod` 확인
